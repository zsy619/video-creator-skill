# 字幕字号修复记录

## 修复时间
2026-05-09

## 问题
- 字幕字号 12px 太小，用户要求调大
- 字幕字号 48px 太大，用户反馈"字幕太小"后又调整为 12px，但用户仍然反馈太小
- 最终确定：**28px** 为竖屏视频（1080×1920）的实测舒适值

## 修复方案
- ASS 字幕 **Fontsize=28**, **PlayResX=1080, PlayResY=1920**
- Alignment=2（底部居中）, MarginV=30
- `ffmpeg -vf "ass=subtitle.ass"` 烧录

## 经验总结
- 竖屏视频（1080×1920）字幕 28px 约等于 1080p 横屏视频的 14-16px
- 12px 对竖屏视频来说太小，48px 对竖屏视频来说太大
- edge-tts 生成的音频需要经过后处理：
  1. `silenceremove` 去静音段
  2. `atempo=1.2` 1.2倍速
  3. `apad=whole_dur=X` 补齐到视频时长
