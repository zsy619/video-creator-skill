# 视频脚本生成系统

> 所属模块：video-creator / SKILL.md → 内容输出

## Markdown 输出结构

```markdown
# 输出到 docs/video-script.md

# 视频标题

## 元数据
- **平台**: 微信视频号 / 小红书
- **时长**: XX秒
- **主题**: [主题名称]

## 小红书版本
### 标题
[吸引眼球的标题，带emoji]

### 正文
[详细内容，100-500字]

### 标签
#标签1 #标签2 #标签3 #标签4 #标签5 #标签6 #标签7

## 视频号版本
### 标题
[简洁有力的标题]

### 描述
[内容摘要，50-150字]

### 话题
#话题1 #话题2 #话题3

## 封面图
![封面图](assets/cover.png)

## 内容插图
![插图1](assets/illustration-1.png)
![插图2](assets/illustration-2.png)

## 视频脚本
### 场景1: 封面 (0-3秒)
- 视觉: [描述]
- 文字: [文字内容]
- 动画: [动画效果]

### 场景2: 内容 (3-20秒)
...
```

## 脚本字段说明

| 字段 | 说明 |
|------|------|
| `场景X` | 场景序号和名称 |
| `时间` | 场景时间范围，如 "0-3秒" |
| `视觉` | 背景、颜色、布局描述 |
| `文字` | 具体显示的文字内容 |
| `动画` | 入场、退场动画效果 |

## 场景分镜示例

```markdown
## 场景1：封面 (0-3秒 | 帧0-180)

副标题：v4 (100px, 青色 #06B6D4)
标签语：持久化任务流来了！ (48px, 白色)

**配音：**
[配音文案]

**动画：**
- 入场：scale 0.85→1 + opacity 0→1，20帧
- 停留：180帧
```
