## 使用方法

### 在代码中引用主题

```javascript
const { THEMES, getTheme, getPlatformSpec } = require('./scripts/themes');

// 获取单个主题
const theme = THEMES['cyberpunk'];

// 或使用辅助函数
const theme = getTheme('tech-modern');

// 获取平台规格
const spec = getPlatformSpec('xiaohongshu');
```

### 在 Remotion 组件中使用

```tsx
import { THEMES } from '../scripts/themes';

const VerticalVideo: React.FC<{ style: string }> = ({ style }) => {
  const theme = THEMES[style] || THEMES['tech-modern'];

  return (
    <AbsoluteFill style={{ backgroundColor: theme.backgroundColor }}>
      {/* 组件内容 */}
    </AbsoluteFill>
  );
};
```

### 在 CLI 中指定主题

```bash
# 使用赛博朋克主题
node cli.js --content "# 标题" --style cyberpunk

# 使用金融科技主题
node cli.js --content "# 标题" --style fintech
```
