# 平台规格

> 所属模块：video-creator / SKILL.md → 平台适配

## 微信视频号

| 属性 | 值 |
|------|-----|
| **分辨率** | 1080x1920 (9:16) |
| **帧率** | 60fps (推荐), 30fps |
| **时长** | 15秒-60秒 (最佳30秒) |
| **文件大小** | < 100MB |
| **编码** | H.264, AAC |
| **码率** | 8-12 Mbps |

## 小红书

| 属性 | 值 |
|------|-----|
| **分辨率** | 1080x1920 (9:16) 或 1440x1920 (3:4) |
| **帧率** | 60fps (推荐), 30fps |
| **时长** | 15秒-5分钟 |
| **文件大小** | < 300MB |
| **编码** | H.264, AAC |

## 抖音/快手

| 属性 | 值 |
|------|-----|
| **分辨率** | 1080x1920 (9:16) |
| **帧率** | 60fps (推荐) |
| **时长** | 15秒-3分钟 |
| **文件大小** | < 200MB |

## Remotion 竖屏配置

```tsx
<Composition
  id="VerticalVideo"
  component={VideoComponent}
  durationInFrames={3600}  // 60秒 @ 60fps
  fps={60}
  width={1080}
  height={1920}
/>
```
